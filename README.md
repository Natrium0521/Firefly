# 流萤工具箱

_使用 [electron](https://github.com/electron/electron) 开发的《崩坏：星穹铁道》工具箱_

</div>

## 目前包含的功能

- 成就管理

- 跃迁记录分析

## 使用

#### 从源代码运行

```
git clone
npm install
npm run main
```

#### 下载打包的文件

从最新 Release 下载压缩包解压运行 Firefly.exe

或下载安装包文件进行安装
