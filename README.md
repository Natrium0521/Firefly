</div>

## 目前包含的功能

- 成就管理

- 跃迁记录分析

## 使用

### 从源代码运行

```
git clone
npm install
npm run main
```

### 下载打包的文件

+ **推荐**：从最新 Release 下载压缩包解压运行 Firefly.exe

+ 或者：下载安装包进行安装
